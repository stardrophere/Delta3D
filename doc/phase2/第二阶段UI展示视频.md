本节对应的UI交互演示视频因文件体积较大（150MB MP4），为优化仓库克隆速度和存储效率，已上传至 GitHub Release 中，具体获取方式如下：

### 📹 视频获取步骤
1. 点击仓库右侧的 **Releases**（或直接访问链接：`https://github.com/android-app-development-course/2025-Autumn-Aberdeen-10-Delta3D/releases/tag/v1.0.0-UI-demo`）
2. 在版本 `v1.0.0-UI-demo` 的 **Assets** 列表中，下载文件名为 `UI_demo.mp4` 的视频文件
3. 视频包含APP所有UI功能的完整交互演示

### 📋 视频内容简介
- 展示UI界面的核心布局与交互逻辑
- 演示关键功能的操作流程
