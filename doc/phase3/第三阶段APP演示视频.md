## 第三阶段APP演示视频

本节对应的APP交互演示视频因文件体积较大（1.04GB ZIP），为优化仓库克隆速度和存储效率，已上传至 GitHub Release 中，具体获取方式如下：

### 📹 视频获取步骤
1. 点击仓库右侧的 **Releases**（或直接访问链接：`https://github.com/android-app-development-course/2025-Autumn-Aberdeen-10-Delta3D/releases`）
2. 在版本 `v1.0.0-app-demo` 的 **Assets** 列表中，下载文件名为 `2K_demo.zip` 的视频压缩文件
3. 视频包含APP所有UI功能的完整交互演示
